Normal 是“下命令”，Insert 才是“打字”
操作 = 操作符 + 移动

光标移动
w/b	下一词/上一词
e	词尾
0/^	
$	
gg/G
C+u/C+d

yy             复制行
yw             复制词
p / P          粘贴（后 / 前）


删除
x              删除字符
dd             删除行
dw / de        删除词
d0 / d$        删除到行首 / 行尾
dG             删除到文件尾

查找跳转
/word          向下查找
?word          向上查找
n / N          下一个 / 上一个
%              匹配括号


u              撤销
Ctrl+r         重做

i              光标前插入
a              光标后插入
I              行首插入
A              行尾插入
o / O          下一行 / 上一行插入
Esc            回到 Normal

v              字符选择
V              行选择
Ctrl+v         块选择

y              复制
d              删除
c              修改
> / <          右移 / 左移缩进

:%s/old/new/g  全文替换
:s/old/new/    当前行替换

:e file        打开文件
:bd            关闭 buffer
Ctrl+w h/j/k/l 切换窗口
